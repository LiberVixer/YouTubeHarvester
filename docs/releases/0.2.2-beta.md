# YouTube Harvester 0.2.2 Beta

## Downloads

- `YouTubeHarvester_0.2.2-beta_linux_all.deb` - Linux Mint / Ubuntu-like systems.
- `YouTubeHarvester_0.2.2-beta_windows_setup.exe` - Windows installer.
- `YouTubeHarvester_0.2.2-beta_windows_x64.msi` - alternative Windows MSI installer.
- `YouTubeHarvester_0.2.2-beta_windows_portable.zip` - portable Windows build.
- `YouTubeHarvester_0.2.2-beta_source.tar.gz` - source archive.

## Notes

- Linux uses the stable Bash downloader by default.
- Windows uses the Python downloader engine only.
- Windows autostart is stored in the Current User registry key:
  `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`.
- The Windows files attached to this beta were built and checked offline on Windows.

## Checksums

Use `SHA256SUMS-linux.txt` and `SHA256SUMS-windows.txt` from the release assets.
