# YouTube Harvester 0.2.4 Beta

## English

This release focuses on making the Windows beta build self-contained and safer to test.

- Windows packages now bundle `ffmpeg.exe`, `ffprobe.exe`, and `deno.exe`.
- The Python downloader passes bundled media tools and the Deno runtime to `yt-dlp`.
- Windows subprocess output is handled as UTF-8, so Cyrillic titles and emoji do not break the app console.
- Downloads that fail on Windows no longer clean the temp directory as if the final file had been moved.
- Output filenames are shortened and use Windows-safe naming rules.
- README, screenshots, offline Windows build notes, and release metadata were refreshed.

Special thanks to Windows beta tester Dmitry 'minion' Pogorily.

Release assets are built by GitHub Actions.

## Русский

Этот релиз доводит Windows beta-сборку до более самодостаточного и безопасного состояния для тестирования.

- Windows-пакеты теперь кладут внутрь `ffmpeg.exe`, `ffprobe.exe` и `deno.exe`.
- Python-движок передаёт bundled media-инструменты и Deno runtime в `yt-dlp`.
- Вывод Windows-процессов читается как UTF-8, поэтому кириллица и emoji не ломают консоль приложения.
- При ошибке скачивания Windows-сборка больше не чистит temp-папку так, будто итоговый файл уже перенесён.
- Имена выходных файлов сокращаются и приводятся к Windows-safe правилам.
- Обновлены README, скриншоты, инструкция офлайн-сборки Windows и метаданные релиза.

Отдельное спасибо бета-тестеру windows версии Дмитрию 'minion' Погорилому.

Бинарники и архив исходников собраны через GitHub Actions.
