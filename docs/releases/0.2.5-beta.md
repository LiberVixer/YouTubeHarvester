# YouTube Harvester 0.2.5 Beta

## English

This release polishes the quick-download workflow, Linux Wayland hotkeys, and the Windows beta experience.

- Added the quick-download window with clipboard URL detection, preview loading, immediate download, queue add, and a persistent Telegram-notification checkbox.
- Added configurable quick-download hotkey support. Windows uses a native global hotkey, Linux/X11 uses `pynput`, and Cinnamon/GNOME Wayland can install a desktop custom shortcut for `yt-harvester --quick-download`.
- The overview screen layout was tightened: controls, counters, archive button, channel logo, log area, and download preview fit more cleanly.
- The channel and video preview images in quick download were corrected and the quick-download window remembers its position.
- Linux now uses the Python downloader engine only; the legacy Bash engine is disabled and documented as obsolete.
- Windows text output and archive display were hardened against mojibake and console encoding problems.
- The Windows build remains self-contained with bundled `ffmpeg.exe`, `ffprobe.exe`, and `deno.exe`.

Special thanks to Windows beta tester Dmitry 'minion' Pogorily.

Release assets are built by GitHub Actions.

## Русский

Этот релиз полирует быстрое скачивание, горячие клавиши Linux Wayland и работу Windows beta-сборки.

- Добавлено окно быстрого скачивания: URL из буфера обмена, предпросмотр, немедленное скачивание, добавление в очередь и сохраняющаяся галочка Telegram-уведомления.
- Добавлена настройка горячей клавиши быстрого скачивания. Windows использует native global hotkey, Linux/X11 — `pynput`, а Cinnamon/GNOME Wayland может сам установить системную пользовательскую команду для `yt-harvester --quick-download`.
- Вкладка обзора стала аккуратнее: кнопки, счётчики, кнопка архива, логотип канала, область логов и предпросмотр скачивания лучше помещаются в окне.
- Исправлены предпросмотры канала и видео в быстром скачивании, окно быстрого скачивания запоминает позицию.
- Linux теперь использует только Python-движок скачивания; устаревший Bash-движок отключён и описан как legacy.
- Отображение текста в Windows и архиве защищено от кракозябр и проблем кодировки консоли.
- Windows-сборка остаётся самодостаточной: внутри есть `ffmpeg.exe`, `ffprobe.exe` и `deno.exe`.
